5. Softmax Regression

Theories

  • All probabilities are nonnegative. We can then transform these values so that they add up to 1 by dividing each by their sum. This process is called normalization.

    $$ \hat{\mathbf{y}} = \mathrm{softmax}(\mathbf{o}) \quad \text{where}\quad \hat{y}_i = \frac{\exp(o_i)}{\sum_j \exp(o_j)} $$

  • Cross Entropy: compare difference between two probability

    $$ H(p, q) = \sum_j - p_i \log (q_i) $$

  • Use Cross Entropy as Loss

    $$ l(\mathbf{y}, \hat{\mathbf{y}}) = - \sum_{j=1}^q y_j \log \hat{y}_j = - \log \hat{y}_y $$

    $$ \partial_{o_j} l(\mathbf{y}, \hat{\mathbf{y}}) = \frac{\exp(o_j)}{\sum_{k=1}^q \exp(o_k)} - y_j = \mathrm{softmax}(\mathbf{o})_j - y_j $$

Loss Functions

  • L2 Loss

    $$ l(y, y’) = \frac{1}{2}(y - y’)^2 $$

    • Sometimes we do not expect large gradient to update parameter when our predicted value is far from the expected value

  • L1 Loss

    $$ l(y, y’) = |y - y’| $$

    • Non-derivative at 0, strong change between -1 and 1 around 0, not stable at the end
    • More stable, provide stable update through training process

  • Huber’s Robust Loss

    $$ l(y, y’) = \begin{cases} |y - y’| - \frac12, & \text{if $|y - y’|>1$}.\ \frac{1}{2}(y - y’)^2, & \text{otherwise}. \end{cases} $$

    • Gradient is y = x from -1 to 1

    def get_fashion_mnist_labels(labels):
    text_labels = ['t-shirt', 'trouser', 'pullover', 'dress', 'coat',

    def load_data_fashion_mnist(batch_size, resize=None):
    trans = [transforms.ToTensor()]
    if resize:
        trans.insert(0, transforms.Resize(resize))
    trans = transforms.Compose(trans)
    mnist_train = torchvision.datasets.FashionMNIST(
        root="../data", train=True, transform=trans, download=True)
    mnist_test = torchvision.datasets.FashionMNIST(
        root="../data", train=False, transform=trans, download=True)
    return (data.DataLoader(mnist_train, batch_size, shuffle=True,
                            num_workers=get_dataloader_workers()),
            data.DataLoader(mnist_test, batch_size, shuffle=False,
                            num_workers=get_dataloader_workers()))

    def softmax(X):
    X_exp = np.exp(X)
    partition = X_exp.sum(1, keepdims=True)
    return X_exp / partition # broadcast

def cross_entropy(y_hat, y):
    return - np.log(y_hat[range(len(y_hat)), y])

  • Accuracy

    def accuracy(y_hat, y):
    if len(y_hat.shape) > 1 and y_hat.shape[1] > 1:
        y_hat = y_hat.argmax(axis=1) # get largest indices on each row
    cmp = y_hat.astype(y.dtype) == y
    return float(cmp.astype(y.dtype).sum()) # correct number of samples

accuracy(y_hat, y) / len(y)

    def evaluate_accuracy(net, data_iter):
    """computer model accuracy on specific dataset"""
		# Appendix A.1
		# length of 2: correct prediction nums, total prediction nums
    metric = Accumulator(2)  
    for X, y in data_iter:
        metric.add(accuracy(net(X), y), d2l.size(y))
    return metric[0] / metric[1]

  • Training

    """training for one epoch"""
def train_epoch_ch3(net, train_iter, loss, updater):
		# Appendix A.1
    # length of 3: sum of training loss、sum of training accuracy、sample nums
    metric = Accumulator(3)
    if isinstance(updater, gluon.Trainer):
        updater = updater.step
    for X, y in train_iter:
        # calculate gradient and update parameters
        with autograd.record():
            y_hat = net(X)
            l = loss(y_hat, y)
        l.backward()
        updater(X.shape[0])
        metric.add(float(l.sum()), accuracy(y_hat, y), y.size)
    # return traning loss and traning accuracy
    return metric[0] / metric[2], metric[1] / metric[2]

# Pytorch does not implicitly modify input shape, so we need to flatten it first
net = nn.Sequential(nn.Flatten(), nn.Linear(784, 10))

def init_weights(m):
    if type(m) == nn.Linear:
        nn.init.normal_(m.weight, std=0.01)

# initialize weight
net.apply(init_weights);

# loss function
loss = nn.CrossEntropyLoss(reduction='none')

# optimization
# net.parameters() - pass all parameters
trainer = torch.optim.SGD(net.parameters(), lr=0.1)

num_epochs = 10
d2l.train_ch3(net, train_iter, test_iter, loss, num_epochs, trainer)
